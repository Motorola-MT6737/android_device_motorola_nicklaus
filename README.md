Device configuration for Motorola Moto E4 Plus ("nicklaus")
===========================================================

The Moto E4 Plus Mediatek (codenamed _"nicklaus"_) is a mid-range smartphone from Motorola.
It was announced on June 2017.

## Device specifications

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | Quad-core 1.3 GHz Cortex-A53
Chipset | MediaTek MT6737
GPU     | Mali-T720
Memory  | 2GB RAM
Shipped Android Version | 7.1.1
Storage | 16GB
MicroSD | Up to 64GB
Battery | Li-Pol 5000mAh battery
Display | 720 x 1280 pixels, 5.5 inches
Camera  | Main 13MP / Front 5MP, autofocus, LED flash
